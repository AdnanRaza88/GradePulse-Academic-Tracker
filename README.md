# GradePulse - AI Powered Academic Performance Tracker

Full Stack Project: FastAPI Backend + Beautiful Streamlit Frontend

## Features
- Grade Management (CRUD)
- Bulk Upload CSV/Excel
- AI Study Tips (Groq)
- AI Daily Routine Planner
- Custom Grading Scales
- Dashboard with Charts

## Folders
- `gradepulse-backend/` → FastAPI API
- `gradepulse-frontend/` → Streamlit UI
