# GradePulse - Complete Project

**AI-Powered Student Grade Management System**

- **Backend**: FastAPI + SQLModel + SQLite + LangChain Groq
- **Frontend**: Streamlit with Neumorphic Design

## Structure
- `gradepulse-backend/` - FastAPI API
- `gradepulse-frontend/` - Streamlit UI

## Deployment on Railway
1. Create two services:
   - Backend service from `gradepulse-backend` folder
   - Frontend service from `gradepulse-frontend` folder
2. Set environment variables (GROQ_API_KEY etc.)
3. Connect volume for DB persistence (backend)
